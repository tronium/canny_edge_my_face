ROS and OpenCV: Canny Edge Filter/Detection
===========================================

Author: Ori Novanda (cargmax-at-gmail.com)

This is an basic demonstration program in using a Webcam, ROS and OpenCV for image processing.

### Input node ###

* /cv_camera/image_raw

### Output node ###

* /camera/canny_edge
* /camera/canny_edge_overlay
